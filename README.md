

# 📝 Blog API Project

A simple RESTful Blog API built with **Node.js**, **Express**, and **MongoDB**.

---

## 🚀 Features

- Create a new blog post
- View all blog posts
- View a single blog post by ID
- Update a blog post
- Delete a blog post

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB with Mongoose
- Postman (for API testing)
- dotenv (for environment variables)

---
